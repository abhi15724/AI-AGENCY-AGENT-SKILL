# 🤖 AI Agency — Multi-Agent System

A collection of specialized AI agents designed to run every function of a lean AI automation agency — from winning clients to delivering projects and managing finances.

Built for solo founders and small AI agencies targeting international clients (US, UK, AU, Canada).

---

## 📁 Agent Overview

| Agent | File | Role |
|---|---|---|
| 🧠 Strategy | `strategy_agent.md` | Niche selection, roadmaps, positioning, growth decisions |
| 📣 Marketing | `marketing_agent.md` | LinkedIn content, case studies, personal brand, content calendar |
| 💼 Sales | `sales_agent.md` | Cold emails, proposals, follow-ups, objection handling |
| 💰 Finance | `finance_agent.md` | Pricing, invoicing, retainer packages, revenue planning |
| 🚀 Delivery | `delivery_agent.md` | Project timelines, SOWs, SOPs, QA checklists, onboarding |

---

## 🧠 Strategy Agent

**File:** `strategy_agent.md`

Provides clear, opinionated business guidance — no vague frameworks.

**Key capabilities:**
- Niche selection using a 3-filter scoring framework (Pain × Pay × Reachability)
- 90-day agency growth roadmap (week-by-week actions)
- Positioning strategy with differentiation angles
- Business decision framework for common agency choices
- Freelancer → Agency transition guide (4 stages)
- Competitive analysis and white space identification

**Best niches scored:** E-commerce (Shopify), Real Estate, SaaS (Series A+), Recruitment

---

## 📣 Marketing Agent

**File:** `marketing_agent.md`

Builds inbound lead flow through content and personal brand.

**Key capabilities:**
- LinkedIn post writing (4 proven formats: Lesson, Result, Contrarian, List)
- 30-day content calendar with weekly rhythm and monthly pillars
- Case study writing (structured template with results-first approach)
- Personal bio writing (short + long versions)
- Twitter/X thread writing
- International client content strategy

**Content pillars:** AI Education · Client Results · Personal Journey · Industry Takes

---

## 💼 Sales Agent

**File:** `sales_agent.md`

Closes deals without sounding like a bot.

**Key capabilities:**
- Cold email writing (Hook → Pain → Tease → CTA, under 150 words)
- Service proposal writing (problem → solution → deliverables → price)
- 5-touch follow-up sequences
- Objection handling (price, timing, in-house, "send more info")
- LinkedIn DM outreach scripts
- Discovery call script (30-min structure, 70/30 listen/talk ratio)

**Core rule:** Sell outcomes, not features. One ask per message.

---

## 💰 Finance Agent

**File:** `finance_agent.md`

Charges what the work is worth and builds predictable revenue.

**Key capabilities:**
- Value-based pricing framework (price at 20–40% of annual value delivered)
- Professional invoice template (USD, international-ready)
- 3-tier retainer package design ($500–$3,000/mo)
- 90-day revenue planning (targets + action steps)
- Upsell strategy (5 methods post-delivery)
- Tax & payment guidance for Indian freelancers earning in USD

**Revenue benchmarks:** $1K/mo = 1 retainer · $5K/mo = 5 retainers or 2 large projects

---

## 🚀 Delivery Agent

**File:** `delivery_agent.md`

Ensures every project runs smoothly and leaves clients delighted.

**Key capabilities:**
- Project timeline creation (7-phase standard AI project structure)
- Scope of Work (SOW) document template
- 48-hour client onboarding process + welcome email template
- SOP document framework (for repeatable tasks and delegation)
- Kickoff call agenda (30-min structure)
- Pre-delivery QA checklist (functionality, integrations, quality, readiness)

**Core rule:** If it's not in the SOW, it's not included.

## 🚀 Getting Started

### Recommended sequence for a new agency:

```
1. Strategy Agent  →  Lock in your niche and 90-day roadmap
2. Marketing Agent →  Set up LinkedIn and start posting
3. Sales Agent     →  Begin cold outreach and discovery calls
4. Finance Agent   →  Set pricing before your first proposal
5. Delivery Agent  →  Use SOW and onboarding docs from day one
```

### How to use each agent:

Each `.md` file is a standalone prompt/skill. Load it into your AI tool of choice (Claude, GPT, etc.) and use it as a system prompt or reference context for that specific function.

---

## 💡 Philosophy

> Each agent operates on its own core principle:

- **Strategy:** Specificity is a superpower. Revenue first, then scale.
- **Marketing:** Inbound > Outbound. Document, don't just create.
- **Sales:** Sell outcomes, not features. Follow up always.
- **Finance:** Price on value, not hours. Retainers are the goal.
- **Delivery:** Clarity prevents conflict. SOPs enable scale.
